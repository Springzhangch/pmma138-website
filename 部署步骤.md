# 香港粤丰展示用品有限公司官网 - 部署步骤

## 第一步：创建 GitHub 仓库

1. **注册/登录 GitHub**
   - 访问 https://github.com
   - 如果没有账号，点击 Sign up 注册

2. **创建新仓库**
   - 登录后，点击右上角 "+" → "New repository"
   - 仓库名称：`pmma138-website`（或你喜欢的名称）
   - 选择 Public（公开）
   - 不要勾选 "Initialize this repository with a README"
   - 点击 "Create repository"

3. **上传网站文件**
   - 在仓库页面，点击 "Add file" → "Upload files"
   - 将以下文件全部拖拽上传：
     - `index.html`
     - `products.html`
     - `about.html`
     - `contact.html`
     - `privacy.html`
     - `robots.txt`
     - `sitemap.xml`
     - `README.md`
   - 点击 "Commit changes" 提交

## 第二步：部署到 Cloudflare Pages

1. **注册/登录 Cloudflare**
   - 访问 https://dash.cloudflare.com/
   - 如果没有账号，点击 Sign Up 注册

2. **将域名添加到 Cloudflare**
   - 在 Cloudflare 控制台，点击 "Add a site"
   - 输入域名：`pmma138.com`
   - 选择免费计划（Free）
   - 点击 "Continue"
   - Cloudflare 会显示两个 NS（名称服务器）地址，例如：
     - `xxx.ns.cloudflare.com`
     - `yyy.ns.cloudflare.com`
   - **重要：** 你需要去你的域名注册商（如阿里云、腾讯云、GoDaddy 等）后台，将域名的 NS 服务器改为 Cloudflare 提供的这两个地址
   - 修改后等待 5-30 分钟生效

3. **创建 Pages 项目**
   - 在 Cloudflare 控制台，点击左侧菜单 "Pages"
   - 点击 "Create a project"
   - 选择 "Connect to Git"
   - 授权 Cloudflare 访问你的 GitHub 账号
   - 选择仓库 `pmma138-website`
   - 点击 "Begin setup"

4. **配置构建设置**
   - 项目名称：`pmma138-website`（或你喜欢的名称）
   - 框架预设：选择 **None**
   - 构建命令：**留空**
   - 构建输出目录：**留空**
   - 根目录：**留空**
   - 点击 "Save and Deploy"

5. **等待部署完成**
   - 部署过程需要 1-3 分钟
   - 完成后会显示一个临时域名，例如：`pmma138-website.pages.dev`
   - 点击这个域名可以预览网站

## 第三步：绑定自定义域名

1. **添加自定义域名**
   - 在 Pages 项目页面，点击 "Custom domains"
   - 点击 "Set up a custom domain"
   - 输入域名：`www.pmma138.com`
   - 点击 "Continue"

2. **配置 DNS**
   - Cloudflare 会自动添加一条 CNAME 记录
   - 如果需要手动配置，在 Cloudflare 控制台：
     - 点击左侧菜单 "DNS"
     - 确保有以下记录：
       - 类型：CNAME
       - 名称：www
       - 目标：`pmma138-website.pages.dev`（或 Cloudflare 提供的地址）
       - 代理状态：启用（橙色云朵）

3. **配置根域名（可选）**
   - 如果你希望 `pmma138.com`（不带www）也能访问：
   - 在 Cloudflare 控制台 → DNS，添加：
     - 类型：A
     - 名称：@（或根域名）
     - IPv4 地址：`192.0.2.1`（占位IP）
     - 代理状态：启用
   - 然后设置 Page Rules：
     - Cloudflare 控制台 → Rules → Page Rules
     - Create Page Rule
     - URL：`pmma138.com/*`
     - 设置：Forwarding URL → 301 Permanent Redirect
     - 目标 URL：`https://www.pmma138.com/$1`

4. **等待 HTTPS 证书**
   - Cloudflare 会自动为你的域名申请 SSL 证书
   - 等待 5-30 分钟，证书会自动激活
   - 证书激活后，访问 `https://www.pmma138.com` 即可

## 常见问题

### Q: DNS 解析不生效？
A: 等待 5-30 分钟，如果超过 1 小时还不生效，检查 NS 记录是否正确。

### Q: HTTPS 证书未激活？
A: 等待 5-30 分钟，Cloudflare 会自动申请证书。如果超过 1 小时，检查 DNS 是否正确。

### Q: 联系表单提交失败？
A: 确认 `contact.html` 的 `action` 中邮箱写对，并已在你邮箱内完成首次提交的确认（FormSubmit 首次提交会发确认邮件）。

### Q: 如何更新网站内容？
A: 在 GitHub 仓库编辑文件，提交后 Cloudflare Pages 会自动重新部署。
